# Threads Post

We open-sourced an AI Interview Coach built with Microsoft Agent Framework.

Upload your resume and a job description, answer behavioral and technical questions, get a performance summary.

It's not one monolithic agent. Five separate agents pass the conversation between each other: receptionist collects your docs, then behavioral interviewer, technical interviewer, and finally a summarizer. A triage agent handles anything that goes off-script.

Built with .NET, Microsoft Foundry, MCP, and Aspire. Deploys to Azure with one command.

https://aka.ms/agentframework/interviewcoach
